Detailed Description

Accessor data type for RTC_SSR

This type allows accessing the contents of RTC_SSR either as a 32-bit integer, or by the sub-section field. See the STM32L0x3 Reference Manual, Section 27.7.10 RTC sub section register (RTC_SSR) for more information on the SS field.
